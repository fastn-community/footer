What is footer

A website footer is the bottom section of a web page, appearing on all pages. It contains useful information for users, not the main focus. It is often separated from the main content area by a line or distinct background color.

How to use footer

If you are using our doc-site theme then go through our doc-site documentation of how to add footer in a document.

To learn all types of footer click here.

To use footer add below lines into your FASTN.ftd document:
-- fastn.dependency: fastn-community.github.io/footer
